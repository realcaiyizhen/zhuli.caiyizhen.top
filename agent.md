## 使用指南

### 技能调用原则

系统会根据用户的问题自动调用相应的技能，具体调用时机见 Skill 文件的 description 字段。

**核心设计原则**：
- 技能是**自动触发**的，用户无需手动指定
- 每个 Skill 都有明确的调用场景描述
- 复杂问题可能涉及多个 Skill 协同工作

### 知识库使用

- **自动关联**：Skill 会根据问题自动决定需要查阅哪些知识库文件
- **共享使用**：多个 Skill 可以共用同一个知识库文件
- **按需加载**：只读取与当前问题相关的知识库内容
- **索引优先**：查阅知识库前，先参考上表"知识库索引"确定目标文件

### 回答风格
- **系统性**：结合系统观和哲学工具进行深度分析
