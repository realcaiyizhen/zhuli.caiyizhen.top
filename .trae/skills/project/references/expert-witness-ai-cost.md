## 2. 技术栈与架构

### 2.1 技术选型

| 层级    | 技术                               | 说明                   |
| ----- | -------------------------------- | -------------------- |
| 前端    | Vue 3 + Vite + Element Plus      | 响应式H5                |
| 后端    | FastAPI + Python 3.12            | REST API             |
| 数据库   | SQLite + SQLAlchemy              | 轻量级本地数据库             |
| 向量存储  | SQLite + sqlite-vec 扩展           | 本地向量检索               |
| 异步任务  | Huey (SqliteHuey)                | 持久化任务队列              |
| 文档转换  | LibreOffice (soffice --headless) | Word/Excel/PPT → PDF |
| PDF处理 | PyMuPDF (fitz)                   | PDF拆页转图片             |
| AI服务  | 阿里云百炼 API                        | 多模态嵌入 + 文本重排序 + AI生成 |
| 用户认证  | FastAPI OAuth2 + JWT             | Token认证              |
| 部署    | Docker/Podman 单容器                | Nginx静态资源 + Uvicorn  |

### 2.2 系统架构

```
前端 (Vue3 H5)
    ↕ HTTP REST
后端 (FastAPI)
    ├── 认证模块 (JWT + OAuth2)
    ├── 案件管理 + 用户授权
    ├── 文件上传处理管线
    │   ├── LibreOffice 文档转换
    │   ├── PyMuPDF PDF拆页
    │   └── 百炼多模态嵌入
    ├── 智能检索 (向量粗排 + AI精排)
    ├── 争议匹配 (AI改写查询 + 证据关联)
    ├── 证据标注
    ├── 案件进度管理
    ├── 批量任务 (断点续传/暂停恢复)
    └── 图片服务
         ↓
    SQLite + sqlite-vec + Huey队列
```
