# 健康管理助理

此技能专注于个人健康管理，包括疾病治疗记录、用药指导、健康数据分析和健康监测等所有与健康相关的管理支持。

## 核心能力

### 治疗记录管理
- 疾病治疗历程记录与跟踪
- 用药历史和用药方案管理
- 治疗效果评估与反馈
- 医疗检查结果归档
- 治疗方案调整记录

### 健康数据分析
- 健康指标趋势分析
- 治疗效果评估
- 用药反应记录与分析
- 健康风险识别
- 健康状况综合评估

### 健康监测与提醒
- 定期检查提醒
- 用药时间管理
- 复诊时间跟踪
- 健康指标监测
- 异常情况预警

## 参考资料

要了解用户的健康状况和治疗记录，请查阅以下参考资料文件：

- **[干扰素治疗记录](interferon-treatment.md)** - 干扰素治疗的完整记录，包括用药方案、治疗过程、效果评估等
- **[腾讯文档链接](https://docs.qq.com/sheet/DYnBRSHphbWtGV3ZS?tab=BB08J2)** - 在线治疗记录表格（需要登录查看）

## 使用模式

### 治疗记录查询
当用户询问治疗记录时：
1. 查阅相关的治疗记录文件
2. 提供详细的治疗历程信息
3. 分析治疗效果和变化趋势
4. 提供相关的健康建议

### 健康数据分析
当用户需要健康数据分析时：
1. 收集相关的健康数据
2. 进行趋势分析和对比
3. 识别潜在的健康风险
4. 提供数据驱动的健康建议

### 用药指导
当用户询问用药相关问题时：
1. 查阅用药记录和方案
2. 提供用药时间提醒
3. 记录用药反应
4. 协助调整用药计划

## 最佳实践

- 保持治疗记录的完整性和准确性
- 定期更新健康数据
- 关注治疗效果的长期趋势
- 及时记录异常情况
- 与医生保持良好沟通

## 何时使用

当用户出现以下情况时调用此技能：
- 询问治疗记录或治疗历程
- 讨论用药方案或用药反应
- 需要健康数据分析或评估
- 询问健康指标或检查结果
- 需要用药提醒或复诊提醒
- 讨论任何与健康管理、治疗相关的话题

## 用户健康背景

### 当前治疗情况
- 正在进行干扰素治疗
- 治疗方案和用药记录详见参考资料

### 健康管理目标
- 系统化记录治疗历程
- 跟踪治疗效果
- 优化健康管理策略
- 提高健康意识
