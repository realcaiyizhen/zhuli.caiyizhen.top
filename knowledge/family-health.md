# 家庭成员健康档案

## 母亲健康档案

### 基本信息
- **出生年份**：1959年
- **年龄**：66岁（2026年）

### 主要健康问题

#### 1. 高血压
- **严重程度**：严重高血压
- **症状**：经常头晕
- **颈动脉检查**：光滑，无斑块
- **特点**：颈动脉光滑但血压高，提示可能为继发性高血压

#### 2. 疑似睡眠呼吸暂停综合征（OSA）
- **打鼾情况**：严重打鼾
- **睡眠质量**：睡眠时间长但质量差
- **其他相关症状**：
  - 易感冒（免疫力下降）
  - 晨起可能血压升高

#### 3. 糖代谢异常
- **空腹血糖**：6.2 mmol/L
- **诊断**：空腹血糖受损（糖尿病前期）

### 症状关联分析

```
严重打鼾 → 睡眠呼吸暂停 → 夜间缺氧
                ↓
    ┌───────────┼───────────┐
    ↓           ↓           ↓
 高血压      胰岛素抵抗   免疫力下降
    ↑           ↓           ↓
 头晕        血糖6.2     易感冒
```

### 待排查项目

| 优先级 | 检查项目 | 目的 |
|--------|----------|------|
| 🔴 高 | 多导睡眠监测（PSG） | 确诊睡眠呼吸暂停综合征 |
| 🔴 高 | 24小时动态血压监测 | 了解血压波动规律，特别是夜间和晨起血压 |
| 🟡 中 | 肾功能 + 肾动脉超声 | 排除肾性高血压 |
| 🟡 中 | 醛固酮/肾素比值 | 筛查原发性醛固酮增多症 |
| 🟡 中 | 糖化血红蛋白 + OGTT | 全面评估糖代谢状态 |
| 🟢 低 | 心脏超声 | 评估高血压对心脏的影响 |

### 初步诊断方向

**最可能诊断**：睡眠呼吸暂停综合征（OSA）导致的继发性高血压

**依据**：
1. 严重打鼾 + 睡眠质量差
2. 高血压伴头晕
3. 颈动脉光滑（排除大血管粥样硬化导致的高血压）
4. 合并糖代谢异常（OSA常见并发症）
5. 免疫力下降（OSA导致深睡眠不足）

### 建议治疗方案

#### 如确诊OSA：
1. **CPAP呼吸机治疗**（首选）
   - 预期效果：血压可下降10-15 mmHg
   - 可能减少或停用降压药
   - 改善血糖控制
   - 提升免疫力和睡眠质量

2. **生活方式干预**
   - 侧卧睡眠
   - 控制体重（如超重）
   - 避免睡前饮酒

#### 血压管理：
- 优先治疗OSA
- 同步调整降压药物
- 考虑ARB类（缬沙坦）或长效CCB（氨氯地平）

#### 血糖管理：
- 生活方式干预
- 控制OSA后血糖可能改善
- 定期监测

### 随访计划

- **近期**：尽快安排睡眠呼吸监测确诊
- **中期**：根据确诊结果制定治疗方案
- **长期**：建立家庭健康监测习惯
